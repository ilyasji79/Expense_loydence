# Loydence Academy Expense Management System

## Overview
Loydence Expense Management System is a web-based application built using PHP and MySQL to manage academy expenses with role-based access control for Admin and HR departments.

## Features
- Admin Dashboard
- HR Dashboard
- Expense Entry & Editing
- Expense Approval Workflow
- Monthly Reports
- PDF Report Generation
- Activity Logs
- Role-Based Authentication
- Responsive Design (Mobile / Tablet / Desktop)

## Technology Stack
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- TCPDF (for PDF generation)

## Installation (Local - XAMPP)

1. Clone repository:
   git clone https://github.com/ilyasji79/Expense_loydence.git

2. Move project to:
   C:\xampp\htdocs\

3. Import database:
   - Open phpMyAdmin
   - Create database: expense_erp
   - Import database.sql

4. Update config:
   config/config.php

5. Start Apache & MySQL in XAMPP

6. Open in browser:
   http://localhost/Expense_loydence/

## Default Roles
- Admin
- HR

## Security Notes
- Protect config files in production
- Use strong database passwords
- Enable HTTPS on live server
